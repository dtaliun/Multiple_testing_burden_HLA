## Data download

This dataset is permanently archived on Zenodo at [doi.org/10.5281/zenodo.21346266](https://doi.org/10.5281/zenodo.21346266)

To download it, use:

`curl -L -o 1K_GWAS_binary.tar.gz https://zenodo.org/records/21346266/files/1K_GWAS_binary.tar.gz?download=1`

Then, unpack the arhcive.
